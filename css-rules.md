**Quick reference:**

| Category        | Prefix      | Example Classes                                                   |
| --------------- | ----------- | ----------------------------------------------------------------- |
| Backgrounds     | `surface-*` | `bg-surface`, `bg-surface-secondary`, `bg-surface-brand`          |
| Borders         | `stroke-*`  | `border-stroke`, `border-stroke-brand`, `border-stroke-error`     |
| Text            | `content-*` | `text-content`, `text-content-secondary`, `text-content-error`    |
| Icons           | `icon-*`    | `text-icon`, `text-icon-brand`, `fill-icon`                       |
| Buttons         | `btn-*`     | `bg-btn-primary`, `border-btn-secondary`, `text-btn-primary-text` |
| Utility         | `util-*`    | `bg-util-blue-500`, `text-util-gray-700`                          |
| Nav             | `nav-*`     | `bg-nav`, `bg-nav-active`                                         |
| Mission Control | `mc-*`      | `bg-mc-modal-bg`, `border-mc-modal-border`                        |

Merge this with the first artifact (primitives, shadows, radii, fonts) for your complete `@theme` block.
